# todolist

## Instalação
1. Clone o repositório do GitHub:

   ```bash
   git clone https://github.com/fabriciosilvaJr/todolist.git
   
2. Navegue até o diretório do projeto:
    `cd todolist`


3. Instale as dependências do Composer:
   `npm install`


### Executando o Aplicativo
```
npm run serve
```
